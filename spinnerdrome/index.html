In July 2020 I created a new word: Spinnerdrome. It's something which reads the same when spun upside down.

It's like a palindrome, but instead of reading the same forwards and backwards, it reads the same if you spin it 180ยบ. Here's an example:

60109

0, 1, and 8 are all valid Spinnerdrome digits. 6 and 9 can be used as a matched pair.

You can also have Spinnerdrome words:

SWIMS

H, I, N, O, S, X, Z are all valid Spinnerdrome letters. M and W can be used as a matched pair.
