Spinnerdrome

I've created a new word; Spinnerdrome. It's a number reading the same when spun upside down.

It's like a palindrome, but instead of reading the same "forwards and backwards" it reads the same if you spin it 180ยบ. Here's an example.

60109

0, 1, and 8 and all valid spinnerdrome digits while 6 and 9 can be used as a pair.
